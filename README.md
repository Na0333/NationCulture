# 🏮 民族文化 · 主题展示网站

> **未来设计师·省三等奖作品**  
> 一个以中国民族文化为主题的静态展示网站。

> 🚀 **在线演示：** [https://wondrous-dango-af7c7e.netlify.app](https://wondrous-dango-af7c7e.netlify.app)

---

## 📖 项目简介

本项目为 **未来设计师大赛 · 省级三等奖** 获奖作品。通过网页多维度展示中国丰富多彩的民族文化，涵盖历史传承、文化遗产、民俗音乐、特色服饰等内容，旨在弘扬中华优秀传统文化。

---

## 📄 页面说明

| 页面 | 说明 |
|------|------|
| [`index.html`](./index.html) | 首页 · 民族文化总览 |
| [`history.html`](./history.html) | 历史传承 · 民族发展脉络 |
| [`heritage.html`](./heritage.html) | 文化遗产 · 非遗与民俗 |
| [`music.html`](./music.html) | 民族音乐 · 传统乐器与歌舞 |
| [`custom.html`](./custom.html) | 特色服饰 · 各民族传统服饰 |

---

## 🛠️ 技术栈

- **HTML5** — 页面结构
- **CSS3** — 布局与样式
- **Bootstrap** — 响应式框架
- **JavaScript** — 交互效果

---

## 🚀 本地运行

```bash
# 直接用浏览器打开
open index.html

# 或使用本地服务器（推荐）
npx serve .
```

---

## 📦 部署

本项目为纯静态页面，支持部署至任意静态托管平台：

- **Netlify** — 拖拽上传即可
- **Vercel / Cloudflare Pages** — 连接 GitHub 自动部署
- **任意 Web 服务器** — 直接放置文件即可
